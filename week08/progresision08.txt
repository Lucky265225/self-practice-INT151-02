dataset
เป็นการจัดเตรียม attribute ที่จะใช้งาน โดยจะเขียน ขึ้นด้วย data- เพื่อนำไปใช้งานต่อใน DOM ในง่ายขึ้น
เช่น <div data-color="red"></div> เข้าถึงโดย div.dataset.color

classList
ช่วยในการเข้าถึง class name ที่จะเข้าไป เพิ่ม เอาออก สลับ ให้ง่ายขึ้น

addEventListener
จับเหตุการณ์ที่เกิดขึ้นใน DOM เพื่อตอบสนองหรือส่งไปทำงานอย่างอื่นต่อ

แบบฝึกหัดที่ทำ
- ช่วยให้เข้าใจโครงสร้างของ HTML มากขึ้น
- ลองวิธีการเข้าถึง node ได้หลายแบบ และเลือกที่เหมาะสมกับการใช้งาน
- ได้เข้าใจวิธีการเพิ่ม tag attribute content ในหน้า HTML แบบไม่ static
- ได้ฝึกการใช้ forEach มากขึ้น
