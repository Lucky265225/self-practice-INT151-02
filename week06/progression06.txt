higher-ORder function คือ ฟังก์ชันที่รับหรือคืนฟังก์ชันอื่นเป็นพารามิเตอร์หรือผลลัพธ์

pure function คือ input ที่รับมาได้เป็น output ไม่ไปกระทบต่อ original value

function composition คือ การเอาผลลัพธ์จากฟังก์ชันหนึ่งไปส่งต่อเป็น input ให้อีกฟังก์ขันหนึ่ง

closure คือ การสร้างสภาพแวดล้อม สำหรับจดจำ และสามารถเข้าถึงตัวแปร scope ภายนอกของมันได้ แม้ว่าจะทำงานเสร็จไปแล้วก็ตาม

ฝึกทำโจทย์
- เข้าใจการเรียก ฟังก์ชันต่อกันมากขึ้น
- การประยุกย์การเก็บ callback ไว้ใน array เราไปใช้เรากับ map ที่ใช้ สำหรับตัวแปรแทน
- closure เราสร้าง ฟังก์ชันสำหรับนับขึ้นมา แต่ก็ ต้องมี ฟังก์ชันแล้วสำหรับ เก็บค่าด้วย
